# Company Profile Open Microlock

Folder ini berisi draft company profile Open Microlock / DARMI.

## File Utama

- `index.html` - halaman company profile statis.
- `styles.css` - styling halaman.
- `script.js` - interaksi preview mobile di hero section.
- `plan.md` - arah konten, branding, dan prioritas.
- `wireframe.md` - gambaran kasar layout desktop dan mobile.

## Cara Buka

Buka file `index.html` langsung di browser.

Path lokal:

```txt

```
